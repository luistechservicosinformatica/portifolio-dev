# 🐍 Pokémon RPG (Terminal)

## 🇧🇷 Português

Um RPG de Pokémon feito em Python, rodando no terminal, com sistema de batalha por turnos, tipos, status, inventário e salvamento de progresso.

---

### 🚀 Funcionalidades

* 🔥 Sistema de batalha por turnos
* ⚡ Tipos de Pokémon (fogo, água, planta, elétrico)
* 🎯 Sistema de vantagem e desvantagem de tipos
* 💥 Dano com variação e crítico
* ☠️ Efeitos de status (burn, poison, stun)
* 📈 Sistema de XP e level up
* 🎒 Inventário (poção e pokebola)
* 🎲 Captura de Pokémon baseada na vida
* 💾 Save automático e manual
* 🎨 Interface com cores (colorama)

---

### 🧠 Como funciona

Você controla um treinador Pokémon pelo terminal.

Ações disponíveis:

* Explorar (encontrar Pokémon)
* Batalhar
* Gerenciar equipe
* Usar itens
* Capturar Pokémon
* Fugir

Sistema de batalha:

* Baseado em turnos
* Dano depende de nível, tipo e aleatoriedade
* Status afetam o combate ao longo do tempo

## 🇺🇸 English

# 🐍 Pokemon RPG (CLI)

A Pokemon RPG built in Python, running in the terminal, featuring a turn-based battle system, types, status effects, inventory, and save system.

---

### 🚀 Features

* 🔥 Turn-based battle system
* ⚡ Pokémon types (fire, water, grass, electric)
* 🎯 Type advantages and weaknesses
* 💥 Damage variation and critical hits
* ☠️ Status effects (burn, poison, stun)
* 📈 XP and level up system
* 🎒 Inventory (potion and pokeball)
* 🎲 Capture system based on enemy HP
* 💾 Auto-save and manual save
* 🎨 Colored terminal interface (colorama)

---

### 🧠 How it works

You control a Pokémon trainer via terminal.

Available actions:

* Explore (find Pokémon)
* Battle enemies
* Manage team
* Use items
* Capture Pokémon
* Run away

Battle system:

* Turn-based
* Damage depends on level, type, and randomness
* Status effects apply over time
