Sobre o sistema

Este é um sistema de login e cadastro desenvolvido com Java (Spring Boot) por Luís Fernando Reis.

Para mais detalhes técnicos sobre a aplicação e o seu funcionamento com uma API, veja o 'README.md' presente no repositório.

Sinta-se em casa para testar absolutamente tudo da aplicação. As rotas, os tratamentos de erros, autenticação por cookie e de brinde ainda pode tentar aplicar uma vulnerabilidade XSS (Cross-Site-Scripting) ou SQL Injection.
